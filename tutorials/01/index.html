Lecture Recap

Functions

  • Combine multiple tasks into a single command
  • We don't care how it does it, we just need to know what it does

Why Write Functions?

  • Break a program into smaller parts
  • Reuse code instead of rewriting it

Steps to Writing a Function

  1. Understand the purpose of the function
  2. Define the data that comes into the function from the caller (in the form of parameters)
  3. Decide on the set of steps that the program will use to accomplish this goal
  4. Compute and return the intended value

Parts of the Function

  1. Name - Describe the purpose of the function
  2. Inputs/ Function parameters - Variables that are passed into the function
  3. Function Body - Your code
  4. Output - Return value of the function

Q6 - Recommended


def is_leap_year(year):
  return (year % 400 == 0) or \
         (year % 4 == 0 and not year % 100 == 0)

Short and elegant solution.

Note the \ is used there to allow code to be written across multiple lines.
