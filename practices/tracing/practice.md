```python

if num % 3 == 0:
  print('num divisible by 3')
elif num % 5 == 0:
  print('num divisible by 5')
elif num % 15 == 0:
  print('num divisible by 3 and 5')
else:
  print('num not divisible by 3 nor 5')
```

###### Q1
```python

  if k % 2:
    i += k
  else:
    j -= k
  k += 1

```

###### Q2
```python
i, k = 100, 3
while i > 0:
  i -= k
  if i % (k + 1):
    k += 1
  else:
    continue
print(i)
```

###### Q3
```python
i = 1
result = 0
while i < 10:
  j = 1
  while j < 100:
    j += 1
    if not (j % i):
      break
  result += j
  i += 1
print(result)
```
